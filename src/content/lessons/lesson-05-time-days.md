Talking about time is fundamental. In Mara, "Noh" refers to the day/sun.

## Days of the Week (Hriq-noh)

| English | Mara | Meaning |
| :--- | :--- | :--- |
| Sunday | **Pakhynoh** | God's Day / Holy Day |
| Monday | **Hriqnoh** | Work Day |
| Tuesday | **Hriqhnôhnoh** | Day after work start |
| Wednesday | **Pakhynoh** | Mid-week (?) |
| Thursday | **Pohchhonoh** | - |
| Friday | **Chutyhnoh** | End of work week |
| Saturday | **Pakhylâ / Hriqnguanoh** | Preparation day / Resting day |

> *Note: Dialects may vary slightly.*

## Time of Day

| English | Mara |
| :--- | :--- |
| Morning | **Mydi** |
| Noon | **Nohchho** |
| Afternoon | **Zâla** |
| Evening | **Zâ** |
| Night | **Zâ** / **Zâ-rhâ** |
| Today | **Atanoh** |
| Tomorrow | **Mylâ** |
| Yesterday | **Zâhnia** |

## Asking "When?"

*   **Khâpa na?** (When?) - *Specific context usually required.*
*   **Atanoh he khâpa noh a châ?** (What day is it today?)
*   **Mylâ he Pakhynoh a châ aw.** (Tomorrow will be Sunday.)

## Practice

1. Look at your calendar. What day is it? Say "**Atanoh he [Day] a châ.**"
2. Say good morning: "**Mydi pha.**"
3. Say good night: "**Zâ pha.**"
